# 📝 Task Manager

A full-stack task management application built with:

- **Backend:** Node.js, Express
- **Frontend:** React
- **Status:** MVP Complete (Create, Read, Update, Delete)

## 🚀 How to run

### Backend
```
cd backend
npm install
npm run dev
```

Frontend
```
cd frontend
npm install
npm start
```
💡 Features
Create, complete, delete tasks

Responsive clean UI

API connected via REST
